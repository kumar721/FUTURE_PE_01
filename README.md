# AI Website Copy System for Local Businesses

## 🚀 Project Overview

Most local businesses lose customers not because their service is poor, but because their website fails to communicate value clearly.

This project demonstrates how **Prompt Engineering + AI tools** can be used to generate **high-converting website copy** — similar to what web agencies offer clients.

The system is designed to be:
- Reusable across multiple business types
- Conversion-focused (not generic AI content)
- Ready for real-world website use

---

## 🏢 Business Case Study

**Business Name:** GlowNest Salon  
**Category:** Beauty Salon  
**Location:** Hyderabad  

### 🎯 Target Audience
- Working professionals (20–45)
- Brides and event clients
- Customers looking for reliable, premium grooming

---

## 🧠 Problem Solved

Local businesses often struggle with:
- Unclear website messaging  
- Weak value propositions  
- Generic or outdated content  
- Poor call-to-actions  

Hiring professional copywriters can be expensive and time-consuming.

This project solves that by creating a **structured AI prompt system** that generates:
- Clear messaging
- Strong CTAs
- Business-specific content

---

## ⚙️ Prompt System Design

The system is modular and reusable.

### 1. Master Prompt
Defines:
- Business type
- Target audience
- Tone
- Output structure

### 2. Homepage Prompt
Generates:
- Headline (value proposition)
- Subheadline
- Intro section
- Key benefits

### 3. Services Prompt
Generates:
- Service names
- What’s included
- Persuasive descriptions

### 4. CTA Prompt
Generates:
- Booking CTAs
- Trust-building CTAs
- Location-based CTAs
